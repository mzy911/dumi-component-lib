## 用法

```js
import { Empty } from 'dumi-component-lib';

// 导入
<Empty />;
```

## API 文档

| 参数      | 说明             | 类型   | 默认值     |
| --------- | ---------------- | ------ | ---------- |
| className | 自定义类名       | string |            |
| text      | 空状态文本       | string | '空空如也' |
| src       | 空状态的图片地址 | string |            |
